## Requirements

Docker

## Run Locally

Clone the project

```bash
  git clone https://github.com/harshthakur1509/kanban
```

Go to the project directory

```bash
  cd kanban
```

Run Docker

```bash
  docker compose up --build
```

Link(using nginx locally)

```bash
  http://localhost
```
